Домашнее задание
Склад

Функционал:
Регистрация позиции на складе.
Добавление определенной позиции на склад
Удаление определённой позиции на склад
Получение остатка на складе.

=============================
Регистрация позиции на складе
Запрашиваемые параметры:
Название позиции. Тип - строка. Обязательный - да.
Количество. Тип - число. Обязательный - да.

Ответ:
Идентификатор новой позиции. Тип - число.
Название позиции. Тип - строка.
Количество. Тип - число.
=============================

Добавление определенной позиции на склад
Запрашиваемые параметры:
Id позиции. Тип - число. Обязательный - да.
Количество. Тип - число. Обязательный - да.

Ответ:
Идентификатор новой позиции. Тип - число.
Название позиции. Тип - строка.
Количество. Тип - число.
=============================

Удаление определённой позиции на склад
Запрашиваемые параметры:
Id позиции. Тип - число. Обязательный - да.
Количество. Тип - число. Обязательный - да.

Ответ:
Идентификатор новой позиции. Тип - число.
Название позиции. Тип - строка.
Количество. Тип - число.
=============================

Получение остатка на складе.
Запрашиваемые параметры:
Ответ:
Список позиций. Тип - массив объектов (id, название позиции, остаток)
Все ответы должны быть в формате Json строки.  Реализация обработки ошибок не требуется.
В качестве БД использовать объект в глобальной области видимости.
